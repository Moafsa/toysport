# Tema Toy Sport - WordPress

## Descrição

Tema moderno e impactante desenvolvido especialmente para a loja de brinquedos Toy Sport. Totalmente otimizado para WooCommerce, SEO e performance.

## Características

- ✅ Design moderno e responsivo
- ✅ Totalmente compatível com WooCommerce
- ✅ SEO otimizado com Schema.org
- ✅ Performance otimizada (lazy loading, minificação)
- ✅ Acessibilidade (ARIA labels, screen reader support)
- ✅ WhatsApp integrado
- ✅ Menu mobile responsivo
- ✅ Breadcrumbs
- ✅ Schema.org para produtos e organização

## Estrutura de Arquivos


## Configuração

1. Instale o tema através do WordPress Admin
2. Configure as opções do tema em **Aparência > Personalizar**
3. Configure o número do WhatsApp
4. Configure o endereço da loja (para Schema.org)
5. Ative o WooCommerce
