## 📊 Fluxo de Funcionamento

### 1. Instalação e Configuração Inicial
```
1. Instalação do plugin
2. Ativação
3. Configuração de credenciais do Mercado Livre
4. Autenticação OAuth
5. Configuração de contas
6. Configuração de sincronização
7. Teste de conexão
```

### 2. Sincronização de Produtos
```
1. Usuário seleciona produtos no WooCommerce
2. Plugin mapeia atributos
3. Envia para API do Mercado Livre
4. Recebe confirmação
5. Atualiza status no WooCommerce
6. Registra no log
```

### 3. Sincronização de Pedidos
```
1. Novo pedido no Mercado Livre
2. Webhook ou polling detecta pedido
3. Plugin cria pedido no WooCommerce
4. Sincroniza dados do cliente
5. Atualiza estoque
6. Envia notificação
```

### 4. Gestão de Mensagens
```
1. Nova pergunta no Mercado Livre
2. Plugin replica no WooCommerce
3. Usuário responde ou usa IA
4. Plugin envia resposta ao Mercado Livre
5. Registra no histórico
```

### 5. Sincronização de Estoque
```
1. Alteração de estoque no WooCommerce
2. Plugin detecta mudança
3. Atualiza no Mercado Livre
4. Confirma atualização
5. Registra no log
```
